# 🌙 LunaBoard

LunaBoard es una aplicación de productividad aesthetic creada con HTML, CSS y JavaScript.

Incluye sistema de tareas dinámico, filtros, categorías, mood tracker, frases motivacionales aleatorias y modo oscuro místico ✨

---

# ✨ Features

- ✅ Añadir y eliminar tareas
- 🌙 Guardado automático con localStorage
- 📚 Categorías personalizadas
- 🔥 Filtros de tareas
- 😊 Mood tracker
- 🌑 Dark mode
- 💫 Frases aleatorias motivacionales
- 📱 Diseño responsive
- ☁️ Glassmorphism UI

---

# 🛠️ Tecnologías

- HTML5
- CSS3
- JavaScript Vanilla
